
## Create Database

```sql
CREATE DATABASE IF NOT EXISTS Day02;
USE Day02;
````

---

## Create Table: Datasets

```sql
CREATE TABLE Datasets (
    dataset_id INT PRIMARY KEY,
    category VARCHAR(20),
    price DECIMAL(8,2)
);
```

---

## Create Table: Sales

```sql
CREATE TABLE Sales (
    sales_id INT PRIMARY KEY,
    dataset_id INT,
    sales_date DATE,
    amount_paid DECIMAL(8,2),
    FOREIGN KEY (dataset_id) REFERENCES Datasets(dataset_id)
);
```

# SQL Practice Queries

## 1. Total Transaction Volume

Find total number of records in the **Sales** table.

```sql
SELECT COUNT(sales_id) AS Total_records
FROM Sales;
```

---

## 2. Total Revenue per Dataset

Calculate total revenue generated by each **dataset_id**.

```sql
SELECT dataset_id, SUM(amount_paid) AS Total_Revenue
FROM Sales
GROUP BY dataset_id;
```

---

## 3. Average Dataset Price per Category

Find average price of datasets within each **category**.

```sql
SELECT category, AVG(price) AS Avg_Price
FROM Datasets
GROUP BY category;
```

---

## 4. Daily Spending Analysis

Find highest and lowest **amount_paid** for each **sales_date**.

```sql
SELECT sales_date,
       MIN(amount_paid) AS Lowest_Sale,
       MAX(amount_paid) AS Highest_Sale
FROM Sales
GROUP BY sales_date;
```

---

## 5. Filter Aggregated Revenue (HAVING)

Find dataset_ids that generated **more than $1000 total revenue**.

```sql
SELECT dataset_id,
       SUM(amount_paid) AS Total_Revenue
FROM Sales
GROUP BY dataset_id
HAVING SUM(amount_paid) > 1000;
```
