## Table Related Queries in SQL

Table related queries are SQL commands used **to create tables, insert data, and retrieve (view) records from tables.**

These queries mainly belong to:
- **DDL** → Creating table structure  
- **DML** → Inserting and viewing data  

---

## CREATE TABLE

Used to **create a new table in a selected database**

### Syntax

```sql
CREATE TABLE table_name (
    column_name1 datatype constraint,
    column_name2 datatype constraint
);
````

### Example: Create Student Table

```sql id="tblcr1"
CREATE TABLE Student (
    RollNo INT PRIMARY KEY,
    Name VARCHAR(50) NOT NULL,
    Marks INT,
    Grade CHAR(1),
    City VARCHAR(50)
);
```

Table structure is created with **columns and constraints**

---

## SELECT (View All Columns)

Used to **retrieve all records from a table**

### Syntax

```sql id="tblsel1"
SELECT * FROM table_name;
```

### Example

```sql id="tblsel2"
SELECT * FROM Student;
```

Displays **all rows and all columns**

---

## INSERT INTO (Add Records)

Used to **insert new rows into a table**

### Syntax

```sql
INSERT INTO table_name (column1, column2)
VALUES (value1, value2);
```

---

## Insert Multiple Rows

```sql id="tblins1"
INSERT INTO Student (RollNo, Name, Marks, Grade, City)
VALUES
(1, 'Ravi', 85, 'A', 'Bangalore'),
(2, 'Anu', 72, 'B', 'Mysore'),
(3, 'Kiran', 90, 'A', 'Bangalore');
```

Multiple records inserted in **single query**

---

## View Specific Columns

```sql id="tblsel3"
SELECT Name, Marks 
FROM Student;
```

Shows only **selected columns**

---

## Interview Key Points

* CREATE TABLE → defines table structure
* SELECT → used to fetch data
* INSERT → used to add new records
* `SELECT *` → returns all columns
* Multiple rows can be inserted using single INSERT
