# Operators in SQL

SQL Operators are used to **perform operations on data** and **filter records** in queries.

They are mainly used in **WHERE clause** and **SELECT statements**.

---

## 1. Arithmetic Operators

Used to perform **mathematical calculations**.

| Operator | Description | Example |
|---------|------------|---------|
| + | Addition | SELECT 10 + 5; |
| - | Subtraction | SELECT 10 - 5; |
| * | Multiplication | SELECT 10 * 5; |
| / | Division | SELECT 10 / 5; |
| % | Modulus (remainder) | SELECT 10 % 3; |

---

## 2. Comparison (Relational) Operators

Used to **compare values**.

| Operator | Description | Example |
|---------|------------|---------|
| = | Equal to | marks = 90 |
| != or <> | Not equal to | marks != 50 |
| > | Greater than | marks > 80 |
| < | Less than | marks < 40 |
| >= | Greater than or equal | marks >= 35 |
| <= | Less than or equal | marks <= 100 |

---

## 3. Logical Operators

Used to **combine multiple conditions**.

| Operator | Description | Example |
|---------|------------|---------|
| AND | Both conditions must be true | marks > 80 AND city = 'Bangalore' |
| OR | Any one condition must be true | marks < 35 OR grade = 'C' |
| NOT | Reverse the condition | NOT marks = 50 |

---

## 4. Special Operators

Used for **specific filtering conditions**.

| Operator | Description | Example |
|---------|------------|---------|
| BETWEEN | Range of values | marks BETWEEN 50 AND 80 |
| IN | Match any value in list | city IN ('Delhi','Mumbai') |
| LIKE | Pattern matching | name LIKE 'A%' |
| IS NULL | Check missing values | marks IS NULL |
| EXISTS | Check subquery result | EXISTS (SELECT * FROM Student) |
