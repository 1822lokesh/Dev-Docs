## Database Related Queries

Database queries are SQL commands used **to create, delete, and view databases and tables present in the DBMS.**

hese commands belong to **DDL (Data Definition Language)** because they manage **database structure.**

---

## CREATE DATABASE

Used to **create a new database**

### Basic Syntax

```sql
CREATE DATABASE db_name;
````

### Safe Syntax (Recommended in Real Time)

```sql
CREATE DATABASE IF NOT EXISTS db_name;
```

### Example

```sql
CREATE DATABASE IF NOT EXISTS CollegeDB;
```

Database will be created **only if it does not already exist**
Prevents **error during execution**

---

## DROP DATABASE

Used to **delete an existing database permanently**

### Basic Syntax

```sql
DROP DATABASE db_name;
```

### Safe Syntax (Recommended)

```sql
DROP DATABASE IF EXISTS db_name;
```

### Example

```sql
DROP DATABASE IF EXISTS CollegeDB;
```

Database will be deleted **only if it exists**
Prevents runtime errors

⚠ This command deletes

* Database
* All tables
* All records

---

## SHOW DATABASES

Used to **display all databases present in MySQL server**

```sql
SHOW DATABASES;
```

---

## USE DATABASE

Used to **select a database to work with**

```sql
USE CollegeDB;
```

---

## SHOW TABLES

Used to **display all tables inside selected database**

```sql
SHOW TABLES;
```
